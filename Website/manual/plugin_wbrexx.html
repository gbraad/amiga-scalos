Scalos Plugin


PLUGIN
  • filename: wbrexx.plugin
  • author: [jurgen@scalos.co.uk]
  • type: [OOP] Window

INSTALLATION
Run the supplied Install Script. -or- Copy the #?.plugin file to the Scalos:Plugins/OOP/ directory.
Start up the main Scalos Preference program and select the 'Plugins' page. Click the 'New' button and then use the file selector icon to the right-end of the string gadget to select you new plugin file from the Scalos:Plugins/OOP/ directory.

DESCRIPTION
Although Scalos has its own arexx port this plugin adds a Workbench 3.5+ compatible AREXX interface to Scalos internals. This allows OS3.5+ arexx scripts to be run under Scalos, producing the same results.

USAGE
Please consult the manual for information on the supported commands and how to use them. The latest manual updates will be available at http://manual.scalos.co.uk. Find the page in the index under "Arexx - Workbench".
