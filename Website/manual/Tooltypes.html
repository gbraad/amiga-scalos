Tooltypes

All icons are checked by Scalos for some special Tooltypes. These allow you to customise the way some icons respond and look on your desktop.

To add a tooltype to your chosen icon, right-click with your mouse, on the desired icon and select 'Information' from the pop-up menu. This will display an information dialog based on the icon you had selected. Locate the tooltypes section of the dialog and add any of the following:

  • SCALOS_NODRAG prevents an icon from having its last snapshot position from being altered. When ever the user drags the icon from its current place, it snaps back to where it was last saved.
  • SCALOS_NOTEXT removes the text label showing the icons filename.
  • SCALOS_NOTOOLTIP prevents a tooltip appearing when the mouse pointer is left over an icon or file name (when viewing in text mode list).

Drawer icons may also have the following tooltype to allow some more unique configuration:

  • SCALOS_PATTERNNO= can be used to assign a specific pattern number (assigned in the Pattern Preferences) to the window which opens upon selecting the drawer icon.
