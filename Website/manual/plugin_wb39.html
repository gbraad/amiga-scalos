Scalos Plugin


PLUGIN
  • filename: wb39.plugin
  • author: Jürgen Lachmann [jurgen@scalos.co.uk]
  • type: [OOP] System Library Patch

INSTALLATION
Copy the #?.plugin file to the Scalos:Plugins/OOP/ directory.
Start up the main Scalos Preference program and select the 'Plugins' page. Click the 'New' button and then use the file selector icon to the right-end of the string gadget to select you new plugin file from the Scalos:Plugins/OOP/ directory.

DESCRIPTION
This plugin provides the missing functionality to enable the use of OS3.5+ tools. The missing functions of the workbench.library v44 are emulated by Scalos instead.
The plugin also adds the OS3.5 WB style volume gauge to the left window border (exactly the same as the volumegauge.plugin for OS3.0/3.1). So users of this plugin do not require volumegauge.plugin to be installed!

You may set gauge preferences using:
  • OS3.5 - "TweakWB" or "WBCtrl" program from Aminet
  • OS3.9 - the supplied "WBPrefs" program.

USAGE
Once installed, the plugin patches the system to provide the new functions in workbench.library. No user interaction is required.
