Arexx

The main Scalos task (the desktop) features a port for sending commands to it via the Arexx scripting system featured in the Amiga OS.

Command Set

  • ABOUT

Further arexx commands are available through the "wbrexx.plugin". This plugin provides a Workbench 3.5+ compatible API. Details of the command set supported can be read here.

For more details on the Arexx scripting language please consult the Arexx links at Amiga University.
