Installation

Before attempting the installation of Scalos please be aware that you will require an installation of MUI with at least version 3.8 which can be found at [Aminet:util/libs/mui38usr.lha] in order to run the configuration programs. It is also sensible to ensure you have a full installation of at least OS3.0 on your boot partition.

To install the main Scalos distribution use the supplied installer. This sets up the directory structure, SCALOS: assign in your s:user-startup sequence and copies relevant files to your system boot partition.

Please consult the Frequently Asked Questions if you have any problems after installation and rebooting to run Scalos.

MUI Custom Classes

The various MUI modules for Scalos make use of an array of custom classes (known as MCCs). It is sensible to have them all installed on your system to avoid problems. Where possible keep them updated when a new release appears on Aminet. Some classes are not posted on the Aminet network, you can find them at "The Unoffical MUI Custom Class List" (TUMULT) http://www.amiga.dk/tumult/tumult.html
