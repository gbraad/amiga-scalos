Scalos Plugin


PLUGIN
  • filename: persist.plugin
  • author: Jürgen Lachmann [juergen_lachmann@t-online.de]
  • type: [OOP] Windows Sub-class

INSTALLATION
Copy the #?.plugin file to the Scalos:Plugins/OOP/ directory.
Copy the Persist.config file to ENVARC:Scalos/ directory.
Start up the main Scalos Preference program and select the 'Plugins' page. Click the 'New' button and then use the file selector icon to the right-end of the string gadget to select you new plugin file from the Scalos:Plugins/OOP/ directory.

DESCRIPTION
Persistant Windows records each directory window you open and close. So should a crash or reboot occur - next time Scalos is started your windows will be re-opened for you.

USAGE
There are two configurable options available, please read the comments in the ENVARC:Scalos/Persist.config file. Default values are provided. Once installed, the plugin functions transparently.
