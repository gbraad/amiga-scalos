What's New In Scalos?

Scalos offers many new, exciting features, in addition to features you may already be familiar with from using Workbench.

Customizable interface components The common modules which make up Scalos can be run in several different GUI styles. This allows the user the choice of how their system will look and feel. Keeping consistency throughout the applications.

Enhanced functionailty The old functions of Workbench have been enhanced to offer many new and useful additions making it easier and nicer to use. The example the Execute Command dialog now offers filename completion, ASL filerequester to select files and much more (MUI version of module only). Full details of all new additions to the common desktop features can be found in the modules specific dosumentation.

4.3gig+ Support Disks with large sizes show up the correct size in the directory windows. No patches required!

Mutitasking desktop Copying a file from one window to another no longer locks up your desktop! You can do multiple things on the desktop whilst they get on with it in the background increasing productivity!

Extended Configurability Your desktop just got more configurable, allowing you to select random backdrops (with scaling), customisable screen and context sensitive menus (with import functionality), pen settings to give your icon text an outline or shadow. Check out the Preferences.

Modular Design Allowing 3rd party developers to add new functionality or replace exisiting components with improved features. See the developers package for details.
