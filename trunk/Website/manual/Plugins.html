Plugins

The Scalos desktop has a neat interface to allow additional functions or replacement functions to be 'bolted on' without the need for hacky patches which are likely to make your system more un-stable. "Plugins" will usually provide a single, or a small selection of related, improvements to the Scalos desktop.


Configuration

The plugins are seperated into two variations. Object Orientated Plugins (OOP) and Menu Plugins. The OOP variety are only covered here. You use the Scalos Main Preference program to add and remove your plugins. This can be found in your Scalos:Prefs folder if you installed to the default locations. Select the 'Plugins' page of the Main Preferences and you will be presented with a similar display to the one below.


To add new plugins to the system:
  • Click the 'New' Button to insert a new plugin
  • The string gadget below will become enabled
  • Use the file icon to the right of the string gadget open a file requester
  • Select the directory Scalos:Plugins/OOP and then the #?.plugin file you want
  • Select 'Save' to store your new plugin preferences.


Note

If you are upgrading a plugin which is already in use (in the current session) it may be required to reboot your machine to flush out the old plugin and run the new one.


Available Plugins

To obtain plugins please visit the official Scalos website then select the 'Download' menu item. You will be presented with a selection of plugins and modules. Plugins which are developed and/or supported by us are documented here. Click on their name to find out more details of their function.


  • title_env - provides additional display attributes to the screen titlebar
  • title_freepens - provides additional display attributes to the screen titlebar
  • ppc_processor - provides additional display attributes to the screen titlebar
  • xtwindows - attaches keyboard control to directory windows
  • wb39 - provides Workbench 3.5+ compatibility
  • pwindows - adds persistent window memory
  • volumegauge - attaches a visual disk capacity indicator to windows
  • devicefilter - hides volume icons from the desktop
  • title_clock - provides additional display attributes to the screen titlebar
