Preferences

Scalos makes use of the old Workbench preference programs and their settings were possible. However, some of the old configuration tools did not offer enough scope and so we have provided new preference programs to bring you many new features. If there is no new preference program listed below, continue to use your existing preference programs located in "Sys:Prefs/".

Please discontinue use of the following preference programs:

  • NewIconsPrefs
  • WBPattern

All new Scalos preference programs currently require MUI to run. Please ensure this is installed before installing the Scalos package if possible. The current preferences documented here are:

  • Scalos Main Preferences
  • Pattern Preferences
  • Palette Preferences
  • Filetypes Plugin Preferences

To configure the look and feel of the context-sensitive popup menus make use of the "PopUpMenu" preference program, usually installed in the "Sys:Prefs/" directory of your computer.
