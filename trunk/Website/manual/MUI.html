MUI - MagicUserInterface

© Copyright 1993-97 by Stefan Stuntz

MUI is a graphic User Interfaces creation and support system. Using a preferences- program the user of a MUI-Application is able to change the appearance of programs according to his personal taste.

MUI is distributed as Shareware. To obtain a complete program-package, including many examples and more information about registering, you should search in magazine cover CDs or the Aminet archive for a file named muiXXusr.lha (XX stands for the latest version number).

The "Scalos" main program needs no MUI, only the preferences programs and certain modules (if chosen during installation) need MUI 3.6 or better.
