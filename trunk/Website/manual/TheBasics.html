The Basics

Getting Started With Scalos

Turn on your computer and let it boot-up. No messing, no hassle Scalos should present itself just like your old Workbench installation did. You may not even notice any difference initally!

Using The Mouse With Scalos

Hold the mouse on a flat surface with the cable extending away from you so that the mouse box rests under the palm of your hand with the buttons under your finger tips. The button onthe left is the selection button, the button on the right is the menu button, and where available the middle mouse button allows you to pan around the contents of a window.

When using the mouse Pointing means moving the mouse so that the tip of the pointer is positioned over an object on the screen. The pointer moves inthe same direction you move the mouse. The mouse can be lifted and repositioned at any time. Lifting the mouse does not move the pointer. Clicking means pressing tand releasing the selection button Double-Clicking means clicking the selection button twice in rapid succession. Double clicking on an icon causes a window to appear or a program to start,. Holiding Down means pressing the mouse button until your action is completed. Dragging means moving screens, windows, and icons by holding down the selection button and moving the mouse.

Selecting An Icon Or Window

Windows are rectablge areas on the screen that can accept display information. Icons are images in windows that represent disks, drawers, projects or tools.

Starting And Quiting A Program

You may already be familiar with this operation, how else did the installer start itself?

To start a program you double click the left mouse button on the applications icon. This should respond with a message in your screen title bar If nothing happens you may have not clicked on the icon quick enough, try again. Do not click your icons many times at once otherwise you may be presented with multiple instances of your chosen program! [grab of a window and selected icon + mouse pointer]

To quit a program you must use the button located in the top left corner, which by default is a small square. [window frame grab with circle around close gadget]

Getting Help

On-line help is essentuial for learning to use the many features of Scalos. There are two kinds of help available, balloon help that gives you a short explination of what is under your mouse, and detailed help on how to do things provided in the on-line help system.

To Get On-line Help

Select the 'About' menu from the screen menu bar. The list of topics appear inthe list in the help window.

Starting A Program Using Execute Command

You can start a program using the "Execute Command" menu option, found under the screen menu 'Desktop'. If you know the directory path of the file or folder you want to open you may enter it in the string gadget [execute command window grab] You may also use the file icon to bring up a file requester to find the file your wanting to run. If you have opened the file or folder previously in this session you will find it easier to select the item from the history box. Click the drop down gadget to revel the list. [grab of execute command history] For more details on the things you can do in the 'Execute COmmand' dialog go to its specific section.

Shutting Down Your Computer

Press the power button (situated on your power supply unit of tower/desktop case) once to turn off the power to your system. You may also need to turn off your TV or monitor seperately.
