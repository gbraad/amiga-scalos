Scalos Plugin


PLUGIN
  • filename: devicefilter.plugin
  • author: David McMinn [dave@scalos.co.uk]
  • type: [OOP] Device List

INSTALLATION
Run the supplied Install Script. -or- Copy the #?.plugin file to the Scalos:Plugins/OOP/ directory.
Start up the main Scalos Preference program and select the 'Plugins' page. Click the 'New' button and then use the file selector icon to the right-end of the string gadget to select you new plugin file from the Scalos:Plugins/OOP/ directory.

OS3.5+ users are encouraged to also install the latest Scalos wb39.plugin to enhance the functionality of the device filtering.
"Ghostbuster" utility can also be removed from the system as the plugin performs the same functions for Scalos.

DESCRIPTION
The plugin hides NDOS and BAD device icons from the Scalos desktop for OS3.0+. If OS3.5+ is installed it is also possible to specifiy any devices to hide from the desktop.

USAGE
OS3.0 / 3.1 users have no control over what devices can be hidden, only BAD and NDOS are removed from the desktop.
OS3.5 users can use "" (available on Aminet) to alter the list of devices to hide.
OS3.9 users can make use of the Workbench preferences program to add additional devices to the list.
