Arexx - Workbench

With the help of the wbrexx.plugin Scalos gains support for a larger Workbench 3.5+ compatible arexx API.

If an arexx command produces an error you will find the error code placed in the WORKBENCH.LASTERROR variable.

Command Set

  • ACTIVATEWINDOW
  • CHANGEWINDOW

For more details on the Arexx scripting language please consult the Arexx links at Amiga University.
