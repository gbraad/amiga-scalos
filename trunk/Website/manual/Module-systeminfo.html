Modules

System Information

The System Information module provides the user with details of the hardware of which Scalos is currently running on. On launching the module you are presented with a window containing three tabs. Each tab is used to divide up infomation about your computer.

Computer

Only the very basic information is listed here for an at a glance check. The machine model number is displayed (including a check for a PPC board). Other information is the motherboard kickstart ROM revision and the total number of megabytes of RAM installed in your machine.

System

A more detailed specification of the attached hardware and software systems currently in use are listed here.

Environment

Environment variables for the current session can be viewed and set using the listview.
